## 7. 누적 학습 패턴 4가지 (registry.json knowledge)

1. **원격 amd64 builder** — M-series Mac에서 QEMU gcc segfault 회피 (`builder.remote: ssh://root@server`)
2. **Vultr 다중 SaaS — kamal-proxy 뒤 nginx vhost 등록 필요** (자동 등록 안 됨)
3. **SQLite + Ahoy jsonb → json** + Avo Resource enum select (PostgreSQL 전용 기능 사용 안 됨)
4. **외장 nosync 디스크 → `git update-index --chmod=+x bin/*` 필수** (권한 보존 안 됨)

추가로 W2~W4 학습:
5. **.kamal/secrets에 placeholder 필수** (deploy.yml env.secret 등록 시 빈 문자열이라도 채워야 함)
6. **wkhtmltopdf 표지 height 100vh 사용 금지** — padding 100mm + page-break-before로 풀페이지
7. **CardComponent는 섹션 톤(light/dark)에 맞춰 색 선택** — slate 가정 컴포넌트가 light 섹션에서 가독성 박살
